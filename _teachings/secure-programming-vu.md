I was a teaching assistant for the course Secure Programming at Vrije Universiteit Amsterdam during the 2024–2025 academic year (Period 3). The course covers secure software development, common vulnerabilities and exploitation techniques, cryptography, memory-error defenses, and web security, with practical assignments in C and C++. During weekly lab sessions I guided students through their group assignment on building a secure chat application in C++ using modern security practices.
